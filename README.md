# 🎮 Hangman Game in Python

> A simple and interactive command-line Hangman game built using Python. Test your vocabulary skills by guessing the hidden word before you run out of chances! 🚀

---

## 📌 Overview

The Hangman Game is a beginner-friendly Python project that randomly selects a word from a predefined list and allows the player to guess it one letter at a time. The player has a limited number of attempts to reveal the complete word.

---

## ✨ Features

✅ Random word selection from a predefined list  
✅ Interactive command-line gameplay  
✅ Displays hidden letters using underscores (_)  
✅ Updates the word after every correct guess  
✅ Tracks remaining attempts  
✅ Win/Loss detection system  
✅ Beginner-friendly Python logic  
✅ Uses loops, conditions, lists, and string manipulation

## 🎯 Game Rules

📌 A random word is selected by the program.

📌 The player guesses one letter at a time.

📌 Correct guesses reveal the letter's position in the word.

📌 Incorrect guesses reduce the remaining turns.

📌 The player wins if all letters are guessed before turns run out.

📌 The player loses when all turns are exhausted.
